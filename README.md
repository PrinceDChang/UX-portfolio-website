# Oey Chang — UX Portfolio

Personal portfolio site for Oey Chang, UX designer and researcher.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Three Fiber (globe)

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Static output is in `dist/` after `npm run build`.
